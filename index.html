Introduction

The Children's Scratch Coding Contest is a competition where you create a creative story, game, or art based on a specific theme. Showcase your imagination and coding skills!

Themes and Evaluation Criteria

Theme: Themes will be announced on the start date.

Evaluation Criteria:

  • Creativity (40%)
  • Code Quality and Structure (30%)
  • Design Completeness (20%)
  • User Experience (10%)

Rules and Examples

1. Participants must use Scratch to create their works.

2. Works must creatively adhere to the competition theme.

3. Submitted works must not copy others' creations.

4. Teams can have up to 3 members.

Competition Schedule

Start Date: January 15, 2024

End Date: February 15, 2024
